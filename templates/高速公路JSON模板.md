# 宁夏公路管理中心吴忠分中心高速公路抽检路段公路技术状况监管分析报告

为及时了解和全面掌握分中心辖养高速公路技术状况，提高养护工作和管理决策的科学性，按照《公路技术状况评定标准》（JTG 5210-2018）相关内容，分中心对部分高速公路进行抽检，以下为抽检结果明细。
